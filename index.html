Featured student projects

Student projects

Final projects from the Python programming section, presented at Projects Day.

Project 1 RavTan Bot

Developer: Amirhossein Norouzi

Anatomy learning assistant

RavTan Bot receives anatomy images, numbers the visible parts, and names each part for the user. Students can also ask follow-up questions about each labeled area.

Project repo
Project 2 MedCardy

Developers: Benjamin Zolfaghari and Mohammad Hossein Kazemi

Medical exam question review

MedCardy helps students create question packs, review previous exams, receive precise answers, and use quick flashcards before exams.

Project website Project repo
Project 3 NeoScreenBot

Developer: Reza Haji Ghasemi

Newborn screening guide

NeoScreenBot checks newborn screening data against the official health ministry guide and returns normal, repeat, confirmatory test, or urgent status.

Project repo
Project 4 BackTestPro

Developer: Mohammad Dehghan Niri

Trading strategy backtest tool

BackTestPro lets users test trading strategies against past data without writing code, then review the results in a short time.

Project repo
Project 5 EurekaAI

Developers: Aline Esmaeili and Elahe Ensani

Digital health system

EurekaAI combines a symptom analyzer, a voice assistant, and a clinic management system for simple medical guidance, voice access, and light patient and appointment management.

Project repo
