# AbzumsAI Pages

I made this repo for the public AbzumsAI site that GitHub Pages serves from `main`.

Open `index.html` in a browser to work locally. It has no build step.

The site links to the current public course repos, the Telegram channel, student projects, teachers, gallery photos, public videos, and the main study material. It also shows the current recovered material counts.

The default view is English. The language switch changes the full page between English and Persian, including page direction.

Site notes live in `docs`, and local checks live in `scripts`.
